# Zenith Fitness Gym

Welcome to the Zenith Fitness Gym website repository. This is a premium, dark-themed, and responsive landing page built for Zenith Fitness in Mysuru. It is designed with modern aesthetics, smooth micro-animations, and high-end visual appeal to WOW users and create a cinematic browsing experience.

## Features

- **Immersive Hero Section**: Features dynamic text glitch effects and a custom HTML5 canvas particle background.
- **Service Cards with Hover Animations**: Interactive cards showcasing our core offerings (Weight Training, CrossFit, Personal Training, Aerobics, and Nutrition Coaching) with smooth gradient hover effects.
- **Masonry Gallery**: A sleek image gallery with an "Instagram-style" hover overlay, demonstrating the premium facilities.
- **Real Google Reviews Slider**: Features authentic feedback from real members styled seamlessly into the dark theme with 5-star ratings and Google badges.
- **Instagram Feed Section**: A dedicated grid section showcasing the latest social media posts and linking directly to our community Instagram page.
- **Contact & Location Info**: Easily accessible contact numbers, timings, and an embedded Google Maps view.
- **Fully Responsive**: Optimized to provide a perfect viewing experience across all devices (Desktop, Tablet, and Mobile).

## Tech Stack

This project was built entirely using vanilla web technologies to maintain high performance and complete customizability:
- **HTML5**
- **CSS3** (with custom CSS Variables, Flexbox, and CSS Grid)
- **Vanilla JavaScript** (for scroll animations, the particle canvas, and the review slider)

## How to Run Locally

You can view this website without installing any dependencies.

1. Clone the repository:
   ```bash
   git clone https://github.com/reddysarath1983-hub/Zenith-Fitness-GYM.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Zenith-Fitness-GYM
   ```
3. Open `index.html` in any modern web browser.
   
   *Alternatively, if you have Python installed, you can run a local server:*
   ```bash
   python -m http.server 8000
   ```
   *Then, open `http://localhost:8000` in your browser.*

## Customization

- **Styling**: Most global styles and brand colors are defined at the top of `style.css` under `:root`. You can easily change `--bg-dark`, `--text-main`, and `--accent-blue` to customize the theme.
- **Assets**: You can replace the placeholder images inside the `assets/` folder with your own high-resolution gym photos. Keep the same file names, or update the source paths in `index.html`.
